# discord-bot-1
Simple Discord Bot

Usage:

Clone repository

run "npm install"

(make a bot in discordapp.com/developers/applications/me)

edit config.json and put in your bot token (must be a bot user)

edit messages.json

run "npm start"

https://discordapp.com/oauth2/authorize?client_id=*CLIENT-ID-HERE*&scope=bot
